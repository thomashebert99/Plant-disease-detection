# Images de demo

Ce dossier contient un petit lot d'images publiees avec le depot pour les demonstrations.

- `plantvillage/` : exemples in-distribution, proches des images propres utilisees pour l'entrainement et le test PlantVillage.
- `plantdoc/` : exemples out-of-distribution, plus proches d'images terrain PlantDoc et utiles pour montrer les limites du modele.

Convention de nommage :

```text
<source>_<species>_<disease>_<index>.jpg
```

Exemple :

```text
plantvillage_tomato_yellow_leaf_curl_virus_01.jpg
```

Les libelles suivent les classes projet normalisees autant que possible :

- `apple_cedar_apple_rust`
- `tomato_yellow_leaf_curl_virus`
- `potato_early_blight`
- `pepper_bacterial_spot`
- `corn_common_rust`
- `grape_black_rot`
